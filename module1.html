Computer Basics & Digital Literacy

Learn the fundamentals of computers and develop essential digital skills

1. Understanding Computers ▶

  • What is a Computer? ▶
    • A computer is an electronic device that processes data and instructions to complete tasks. It can perform complex calculations, store vast data, and connect to networks. At its core, it processes user input and produces output like text or images.
  • Types of Computers ▶
    • Computers vary from large desktops, laptops, tablets to smartphones. Each type serves different purposes but operates on consistent principles.
  • Basic Parts of a Computer ▶
    • CPU: The brain that processes instructions.
    • Monitor: Displays information visually.
    • Keyboard: Device to input text.
    • Mouse: Pointing device for navigation.
    • Storage Devices: Save files long-term (HDD, SSD, USB).

2. Operating Systems ▶

  • What is an Operating System (OS)? ▶
    • OS manages hardware and software resources and provides services for computer programs.
  • Popular OS Examples ▶
    • Windows, macOS, Linux, and Android are popular operating systems.
  • Functions of an OS ▶
    • Managing files, running applications, controlling devices, and providing user interface.

3. Using Basic Software ▶

  • Word Processors ▶
    • Programs like Microsoft Word and Google Docs let you create and format text documents.
  • Spreadsheets ▶
    • Programs like Excel and Google Sheets help organize and analyze data.
  • Presentation Software ▶
    • Tools like PowerPoint and Google Slides create slide shows for meetings and lessons.

4. File Management ▶

  • Creating & Organizing Files ▶
    • Save files with proper names and group them in folders for easy access.
  • File Extensions ▶
    • File extensions (.docx, .pdf, .jpg) indicate the file type and software to open it.
  • Recycle Bin & Backup ▶
    • Use the Recycle Bin to recover or delete files permanently. Back up important data to external drives or cloud.

5. Introduction to the Internet ▶

  • What is the Internet? ▶
    • The internet is a global network connecting millions of devices, enabling communication and data sharing.
  • Web Browsers ▶
    • Programs like Chrome or Firefox let you access websites, watch videos, and use web applications.
  • Email Basics ▶
    • Creating email accounts, sending/receiving messages, attaching files, and online etiquette.

6. Digital Communication Tools ▶

  • Messaging Apps ▶
    • Apps like WhatsApp and Messenger enable quick text and media communication.
  • Video Calling ▶
    • Tools like Zoom, Google Meet, or Skype connect people face-to-face virtually.
  • Social Media ▶
    • Share updates, photos, and join community groups responsibly on platforms like Facebook.

7. Internet Safety and Security ▶

  • Creating Strong Passwords ▶
    • Use complex passwords with letters, numbers, and symbols to protect your accounts.
  • Recognizing Scams ▶
    • Be aware of phishing emails and suspicious websites that try to steal your info.
  • Software Updates & Antivirus ▶
    • Keep your system and antivirus software updated to protect against malware.
  • Safe Browsing Habits ▶
    • Avoid clicking unknown links or downloading files from untrusted sources.

8. Productivity and Cloud Tools ▶

  • Cloud Storage ▶
    • Services like Google Drive and OneDrive let you save and access files online from anywhere.
  • Real-time Collaboration ▶
    • Work with others on documents simultaneously improving teamwork and productivity.
  • Productivity Tools ▶
    • Use online calendars, notes, and task management apps to organize your learning and projects.
